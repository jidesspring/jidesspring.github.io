图形平移，旋转，缩放变换

发表于 2019-06-03

图形的旋转，平移，缩放其实都是对图形的坐标进行相应的矩阵运算得到的。首先要了解矩阵的基础运算知识：
1.矩阵的加法，和减法，都是对应位置的数相加减
2.矩阵的乘法：矩阵A乘以矩阵B要求矩阵A的列数等于矩阵B的行数，然后每行的元素和每列的元素依次相乘的和产生新矩阵的每个元素
3.矩阵的除法：其实就是乘以被除矩阵的逆矩阵。 根据公式：矩阵*乘以它的逆矩阵等于单位矩阵，可以算出逆矩阵
